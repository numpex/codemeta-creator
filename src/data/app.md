# Codemeta Packager

This is a small webpage to help you get started with a first basic
codemeta specifically tuned for the NumPEX software catalog.

## Usage

It offers a example template based on the StarPU software.

The purpose is just to give you a working example out of the box.

To save your codemeta.json file, just click on the button "DOWNLOAD
JSON" in the upper right corner.

Currently there is no way to save/import a recipe, if you refresh the
page the data will be reset to the default one!
